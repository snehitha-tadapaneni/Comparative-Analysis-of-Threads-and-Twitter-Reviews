# Data Folder

This folder contains datasets used for the project.

## Files
- `threads_reviews.csv` – Threads reviews dataset downloaded from kaggle. This file includes reviews in 2023.
- `twitter_2023-07.csv` – Twitter reviews dataset downloaded from Kaggle. This file only includes the timeframe July, 2023.


## Source
Original dataset: 
- Threads Reviews Dataset: [Kaggle - Threads Reviews](https://www.kaggle.com/datasets/saloni1712/threads-an-instagram-app-reviews/data)
- Twitter Reviews Dataset: [Kaggle - Twitter Reviews](https://www.kaggle.com/datasets/bwandowando/2-million-formerly-twitter-google-reviews)
